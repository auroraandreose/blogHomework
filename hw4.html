Application: Random Walk scalata e convergenza al Moto Browniano

In questa sezione mostro come una Random Walk discreta, costruita con passi di Rademacher (+1 e -1) e opportunamente riscalata per 1 / √n, tenda al Moto Browniano Aritmetico al crescere del numero di passi.

Codice utilizzato

Di seguito riporto una parte essenziale del codice JavaScript utilizzato per generare la Random Walk con passi di Rademacher e costruire il processo riscalato.

JavaScript


function generateRademacher() {
  return Math.random() < 0.5 ? -1 : 1;
}

function simulateScaledRandomWalk(n, T) {
  const times = [];
  const rwValues = [];
  let partialSum = 0;

  times.push(0);
  rwValues.push(0);

  for (let k = 1; k <= n; k++) {
    const step = generateRademacher();
    partialSum += step;

    const scaledRW = partialSum / Math.sqrt(n);

    times.push((k * T) / n);
    rwValues.push(scaledRW);
  }

  return { times, rwValues };
}

Theory: Principio di Invarianza di Donsker

Il Principio di Invarianza di Donsker, noto anche come teorema del limite funzionale, afferma che una Random Walk discreta con incrementi indipendenti, media nulla e varianza finita, se opportunamente riscalata, converge al Moto Browniano Aritmetico quando il numero di passi tende all’infinito.

Nel nostro caso, partendo da variabili di Rademacher, cioè passi che assumono valore +1 oppure -1 con uguale probabilità, definiamo il processo

X_n(t) = (1 / √n) · Σ ξ_i

con la somma estesa ai primi k passi, dove t = kT/n.

Il fattore 1 / √n è fondamentale perché stabilizza la varianza del processo. Senza questo riscalamento, al crescere di n la variabilità aumenterebbe troppo rapidamente. Con questa normalizzazione, invece, il processo discreto assume una forma sempre più simile a quella di una traiettoria browniana.

Il significato profondo del principio di invarianza è che il limite non dipende dalla distribuzione specifica dei passi, purché siano soddisfatte alcune condizioni essenziali: indipendenza, media zero e varianza finita. Per questo si parla di proprietà di invarianza.

In termini intuitivi, aumentando il numero di suddivisioni temporali, la Random Walk scalata perde il carattere fortemente discreto e mostra un comportamento sempre più regolare dal punto di vista globale, avvicinandosi al Moto Browniano Aritmetico.
