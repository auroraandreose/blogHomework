1. Application: Manipolazioni del DOM

In questa sezione dimostro come JavaScript possa interagire con gli elementi della pagina tramite script esterno.

Elemento target per le manipolazioni.

Codice Utilizzato

HTML 
<div id="box">
    <p id="testo-dinamico">Elemento target per le manipolazioni.</p>
</div>

<div class="controls">
    <button onclick="cambiaContenuto()">Cambia Testo</button>
    <button onclick="cambiaColore()">Cambia Colore</button>
    <button onclick="aggiungiElemento()">Aggiungi Elemento</button>
    <button onclick="resetBox()">Reset</button>
</div>

<!-- Collegamento al file esterno -->
<script src="script.js"></script>
JavaScript 
// 1. Manipolazione del CONTENUTO
function cambiaContenuto() {
    const testo = document.getElementById("testo-dinamico");
    testo.innerHTML = "Testo modificato! Ora il DIV contiene nuovi tag HTML.";
    console.log("Contenuto cambiato.");
}

// 2. Manipolazione dello STILE (CSS)
function cambiaColore() {
    const box = document.getElementById("box");
    const randomColor = Math.floor(Math.random()*16777215).toString(16);
    box.style.backgroundColor = "#" + randomColor;
    box.style.border = "3px solid #1a1a1a";
    box.style.borderRadius = "20px";
}

// 3. Manipolazione della STRUTTURA (Aggiunta di nuovi nodi)
function aggiungiElemento() {
    const box = document.getElementById("box");
    const nuovoParagrafo = document.createElement("p");
    nuovoParagrafo.innerText = "✨ Nuovo elemento creato alle " + new Date().toLocaleTimeString();
    nuovoParagrafo.style.color = "#000000";
    box.appendChild(nuovoParagrafo);
}

// 4. RESET (Ripristino dello stato iniziale)
function resetBox() {
    const box = document.getElementById("box");
    box.innerHTML = '
Elemento target per le manipolazioni.
';
    box.style.backgroundColor = "white";
    box.style.border = "1px solid #ddd";
    box.style.borderRadius = "8px";
    box.style.padding = "25px";
}

2. Theory: Relazione di ricorrenza di Welford / Knuth

La relazione di Welford è un metodo iterativo per calcolare la varianza aggiornando media e devianza ad ogni nuova osservazione. Questo approccio evita l'instabilità numerica e i problemi di approssimazione tipici dei metodi tradizionali, garantendo precisione ed efficienza in contesti di analisi statistica e calcolo numerico.

Aggiornamento della media

Sia \(\bar{x}_n\) la media di \(n\) osservazioni. Possiamo scriverla in funzione della media precedente \(\bar{x}_{n-1}\):

\[ \bar{x}_n = \frac{\sum_{i=1}^{n} x_i}{n} = \frac{x_n + \sum_{i=1}^{n-1} x_i}{n} \]

Sapendo che \(\sum_{i=1}^{n-1} x_i = (n-1)\bar{x}_{n-1}\), sostituendo otteniamo:

\[ \bar{x}_n = \frac{x_n + (n-1)\bar{x}_{n-1}}{n} = \bar{x}_{n-1} + \frac{x_n - \bar{x}_{n-1}}{n} \]

Aggiornamento della devianza (\(SS_n\))

Vogliamo trovare una relazione ricorsiva per la somma dei quadrati degli scarti \(SS_n = \sum_{i=1}^{n} (x_i - \bar{x}_n)^2\). Partiamo separando l'ultimo termine:

\[ SS_n = \sum_{i=1}^{n-1} (x_i - \bar{x}_n)^2 + (x_n - \bar{x}_n)^2 \]

Per i termini della sommatoria, applichiamo l'artificio di aggiungere e sottrarre \(\bar{x}_{n-1}\):

\[ (x_i - \bar{x}_n) = (x_i - \bar{x}_{n-1}) + (\bar{x}_{n-1} - \bar{x}_n) \]

Elevando al quadrato ed espandendo il binomio:

\[ \sum_{i=1}^{n-1} (x_i - \bar{x}_n)^2 = \sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1})^2 + \sum_{i=1}^{n-1} (\bar{x}_{n-1} - \bar{x}_n)^2 + 2(\bar{x}_{n-1} - \bar{x}_n) \sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1}) \]

Proprietà fondamentale: Poiché la somma degli scarti dalla propria media è nulla, \(\sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1}) = 0\), il termine centrale scompare. Otteniamo:

\[ \sum_{i=1}^{n-1} (x_i - \bar{x}_n)^2 = SS_{n-1} + (n-1)(\bar{x}_{n-1} - \bar{x}_n)^2 \]

Sostituendo questa espressione nella formula iniziale di \(SS_n\) e utilizzando la relazione di aggiornamento della media \(\bar{x}_n - \bar{x}_{n-1} = \frac{x_n - \bar{x}_{n-1}}{n}\), dopo alcune manipolazioni algebriche arriviamo alla forma finale:

\[ SS_n = SS_{n-1} + (x_n - \bar{x}_{n-1})(x_n - \bar{x}_n) \]

Conclusione

La varianza campionaria si ottiene infine dividendo la devianza per i gradi di libertà: \(s^2 = \frac{SS_n}{n-1}\).
