1. Application: Varie Manipolazioni del DOM

Il DOM (Document Object Model) è l'interfaccia di programmazione che permette a JavaScript di accedere e manipolare dinamicamente il contenuto, la struttura e lo stile dei documenti HTML.

Elemento target per le manipolazioni.

// Esempio di manipolazione della struttura in JS:

const nuovo = document.createElement("p");
nuovo.innerText = "Nuovo elemento";
box.appendChild(nuovo);

Theory: Algoritmo di Welford

L’algoritmo di Welford permette di aggiornare la media e la varianza in modo incrementale senza salvare tutti i dati.

μₙ₊₁ = μₙ + (xₙ₊₁ − μₙ)/(n+1)
