1. Application: Manipolazioni del DOM

In questa sezione dimostro come JavaScript possa interagire con gli elementi della pagina tramite script esterno.

Elemento target per le manipolazioni.

Codice Utilizzato

HTML (Struttura)
<div id="box">
    <p id="testo-dinamico">Elemento target per le manipolazioni.</p>
</div>

<div class="controls">
    <button onclick="cambiaContenuto()">Cambia Testo</button>
    <button onclick="cambiaColore()">Cambia Colore</button>
    <button onclick="aggiungiElemento()">Aggiungi Elemento</button>
    <button onclick="resetBox()">Reset</button>
</div>

<!-- Collegamento al file esterno richiesto -->
<script src="script.js"></script>
JavaScript (Logica nel file script.js)
function cambiaContenuto() {
    const testo = document.getElementById("testo-dinamico");
    testo.innerHTML = "Testo modificato!";
}

function cambiaColore() {
    const box = document.getElementById("box");
    const randomColor = Math.floor(Math.random()*16777215).toString(16);
    box.style.backgroundColor = "#" + randomColor;
    box.style.border = "3px solid #1a1a1a";
}

function aggiungiElemento() {
    const box = document.getElementById("box");
    const nuovoParagrafo = document.createElement("p");
    nuovoParagrafo.innerText = "Nuovo elemento aggiunto!";
    box.appendChild(nuovoParagrafo);
}

function resetBox() {
    location.reload();
}

2. Theory: Algoritmi Online e Metodo di Welford / Knuth

Gli algoritmi online permettono di aggiornare una statistica quando arriva una nuova osservazione senza dover rielaborare tutti i dati precedenti.

Obiettivo della dimostrazione

Mostrare come aggiornare la devianza $SS_n$ (somma degli scarti quadratici) partendo dal valore precedente $SS_{n-1}$ quando si aggiunge $x_n$:

$$SS_n = SS_{n-1} + (x_n - \bar{x}_{n-1})(x_n - \bar{x}_n)$$

Proprietà utilizzate

  • Definizione di devianza: $SS_n = \sum_{i=1}^{n} (x_i - \bar{x}_n)^2$
  • Somma degli scarti nulla: $\sum_{i=1}^{n} (x_i - \bar{x}_n) = 0$
  • Aggiornamento della media: $\bar{x}_n = \bar{x}_{n-1} + \frac{x_n - \bar{x}_{n-1}}{n}$

Dimostrazione e Manipolazioni

Partiamo dalla scomposizione della devianza al passo $n$. Applichiamo un artificio algebrico aggiungendo e sottraendo $\bar{x}_{n-1}$ all'interno del quadrato:

$$SS_n = \sum_{i=1}^{n} [(x_i - \bar{x}_{n-1}) + (\bar{x}_{n-1} - \bar{x}_n)]^2$$

Sviluppando il quadrato del binomio, il termine del doppio prodotto si annulla grazie alla proprietà della somma degli scarti nulla rispetto alla media. Rimangono solo i termini che definiscono la devianza precedente e l'incremento dovuto al nuovo dato.

Utilizzando la relazione di aggiornamento della media e raccogliendo a fattor comune, si ottiene la formula ricorsiva finale di Welford, che garantisce stabilità numerica ed efficienza computazionale.

Calcolo finale della Varianza:
Una volta ottenuta $SS_n$, la varianza campionaria è semplicemente: $$s^2 = \frac{SS_n}{n-1}$$
