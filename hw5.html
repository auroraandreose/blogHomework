Codice utilizzato

La simulazione parte dalla generazione di una traiettoria browniana discreta tramite una Random Walk scalata. Su questa traiettoria viene poi costruito il Geometric Brownian Motion (GBM), utilizzando i parametri di drift \( \mu \), volatilità \( \sigma \) e valore iniziale \( S_0 \).

JavaScript

function generateRademacher() {
  return Math.random() < 0.5 ? -1 : 1;
}

function simulateBrownianPath(n, T) {
  const times = [0];
  const brownian = [0];
  let partialSum = 0;

  for (let k = 1; k <= n; k++) {
    partialSum += generateRademacher();
    times.push((k * T) / n);
    brownian.push(partialSum / Math.sqrt(n));
  }

  return { times, brownian };
}

function buildGBM(times, brownian, mu, sigma, s0) {
  const gbm = [];

  for (let i = 0; i < times.length; i++) {
    const t = times[i];
    const w = brownian[i];

    gbm.push(s0 * Math.exp((mu - 0.5 * sigma * sigma) * t + sigma * w));
  }

  return gbm;
}

Theory: ABM, GBM e processi a Mean Reversion

L’Arithmetic Brownian Motion (ABM) è uno dei modelli continui più semplici per descrivere un processo stocastico. Si scrive come

\( X_t = X_0 + \mu t + \sigma W_t \)
dove \( \mu \) rappresenta il drift, \( \sigma \) la volatilità e \( W_t \) il Moto Browniano.

In questo modello gli incrementi sono normali e indipendenti, e il processo evolve in modo additivo. Per questo motivo, l’ABM può assumere anche valori negativi ed è particolarmente utile come modello teorico di base.

Il Geometric Brownian Motion (GBM) è invece ottenuto imponendo una dinamica moltiplicativa, ed è il modello classico per descrivere l’evoluzione dei prezzi in finanza. La sua espressione è:

\( S_t = S_0 \exp\left[\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma W_t\right] \)
Il GBM resta sempre positivo, poiché è costruito come esponenziale di un processo gaussiano.

Rispetto all’ABM, il GBM è più adatto a rappresentare grandezze che non possono diventare negative, come i prezzi azionari. Inoltre, i rendimenti logaritmici risultano normalmente distribuiti.

Un’altra famiglia importante di processi stocastici è quella dei processi a mean reversion, nei quali la dinamica tende a riportare il processo verso un livello di equilibrio di lungo periodo.

Il caso più noto è il processo di Ornstein–Uhlenbeck, descritto da:

\( dX_t = \theta(\mu - X_t)\,dt + \sigma\,dW_t \)
dove \( \theta > 0 \) misura la velocità di ritorno verso la media \( \mu \).

In questo caso, quando \( X_t \) si allontana dalla media di lungo periodo \( \mu \), il termine \( \theta(\mu - X_t) \) esercita una forza di richiamo che tende a riportarlo verso tale livello. Questo comportamento è molto diverso da quello del Moto Browniano, che non presenta alcun meccanismo di rientro.

In sintesi, l’ABM descrive un’evoluzione additiva, il GBM una dinamica moltiplicativa e sempre positiva, mentre il processo di Ornstein–Uhlenbeck rappresenta un modello con ritorno verso la media. Questi tre esempi mostrano come piccole variazioni nella struttura dell’equazione stocastica producano comportamenti qualitativamente molto differenti.
