Parte B - Generazione di distribuzioni da U[0,1)

Dato un generatore uniforme in [0,1), molte variabili casuali possono essere ottenute applicando opportune trasformazioni matematiche. L'idea di fondo è che il generatore uniforme costituisce il punto di partenza per costruire distribuzioni continue e discrete di interesse statistico.

Principali metodi di generazione

  • Metodo della trasformata inversa: se la funzione di ripartizione F è invertibile, si genera X = F-1(U).
  • Metodo di composizione: una variabile complessa può essere costruita combinando variabili più semplici già generate.
  • Metodo di accettazione-rifiuto: si genera da una distribuzione ausiliaria e si accetta il valore con una certa probabilità, in modo da ottenere la distribuzione desiderata.
  • Trasformazioni tra distribuzioni note: alcune distribuzioni si ottengono direttamente da altre, ad esempio come somme, rapporti o trasformazioni di variabili Normali, Gamma o Chi-quadro.

Algoritmi e formule per le principali distribuzioni

  • Uniforme U[0,1): X = U
  • Discreta arbitraria: si costruiscono le probabilità cumulative Cj = p1 + ... + pj e si pone X = xj se Cj-1 ≤ U < Cj
  • Bernoulli(p): X = 1 se U < p, altrimenti X = 0
  • Binomiale B(n,p): X = X1 + ... + Xn, con Xi Bernoulli(p) indipendenti
  • Poisson(λ): algoritmo di Knuth: si moltiplicano uniformi finché il prodotto scende sotto exp(-λ)
  • Esponenziale(λ): X = -ln(1-U) / λ
  • Weibull(k,λ): X = λ[-ln(1-U)]^(1/k)
  • Pareto(α, xm): X = xm(1-U)^(-1/α)
  • Cauchy(x0,γ): X = x0 + γ tan[π(U - 1/2)]
  • Triangolare(a,b,c): distribuzione generata con trasformata inversa definita a tratti
  • Normale standard N(0,1): metodo di Box-Muller: Z1 = sqrt(-2 ln U1) cos(2πU2), Z2 = sqrt(-2 ln U1) sin(2πU2)
  • Normale N(μ,σ²): X = μ + σZ, con Z normale standard
  • Gamma(α,β): in generale si usa un algoritmo di accettazione-rifiuto o metodi specifici come Marsaglia-Tsang
  • Beta(α,β): se G1 ~ Gamma(α,1) e G2 ~ Gamma(β,1), allora X = G1 / (G1 + G2)
  • Chi-quadro(ν): X = Z1² + ... + Zν², con Zi Normali standard indipendenti
  • Student(tν): T = Z / sqrt(V/ν), con Z ~ N(0,1) e V ~ Chi-quadro(ν)
  • Fisher F(ν1,ν2): F = (V1/ν1) / (V2/ν2), con V1 e V2 Chi-quadro indipendenti
