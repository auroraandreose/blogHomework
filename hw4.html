Application: Random Walk scalata e convergenza al Moto Browniano

In questa sezione mostro come una Random Walk discreta, costruita con passi di Rademacher (+1 e -1) e opportunamente riscalata per 1 / √n, tenda al Moto Browniano Aritmetico al crescere del numero di passi.

Codice utilizzato

Riporto di seguito la parte essenziale del codice JavaScript utilizzato per implementare la simulazione. In particolare, il codice genera i passi di Rademacher, costruisce la somma parziale e applica il riscalamento 1 / √n per ottenere il processo discreto da confrontare con il comportamento browniano.

JavaScript

function generateRademacher() {
  return Math.random() < 0.5 ? -1 : 1;
}

function simulateScaledRandomWalk(n, T) {
  const times = [];
  const rwValues = [];
  let partialSum = 0;
  let plusCount = 0;
  let minusCount = 0;

  times.push(0);
  rwValues.push(0);

  for (let k = 1; k <= n; k++) {
    const step = generateRademacher();
    partialSum += step;

    if (step === 1) {
      plusCount++;
    } else {
      minusCount++;
    }

    const scaledRW = partialSum / Math.sqrt(n);

    times.push((k * T) / n);
    rwValues.push(scaledRW);
  }

  return { times, rwValues, plusCount, minusCount };
}

Theory: Principio di Invarianza di Donsker

Il Principio di Invarianza di Donsker, noto anche come teorema del limite funzionale, afferma che una Random Walk discreta con incrementi indipendenti, media nulla e varianza finita, se opportunamente riscalata, converge al Moto Browniano Aritmetico quando il numero di passi tende all’infinito.

Nel nostro caso consideriamo una successione di variabili di Rademacher, cioè variabili che assumono i valori +1 e −1 con uguale probabilità. Se indichiamo con \(\xi_1, \xi_2, \dots, \xi_k\) i primi incrementi, il processo riscalato si può scrivere come:

\( X_n(t) = \frac{1}{\sqrt{n}} \sum_{i=1}^{k} \xi_i \)
con \( t = \frac{kT}{n} \), per \( k = 0,1,\dots,n \).

Il fattore \( \frac{1}{\sqrt{n}} \) è fondamentale perché consente di stabilizzare la varianza della somma parziale. Senza questa normalizzazione, la variabilità crescerebbe troppo rapidamente all’aumentare di \( n \); con il riscalamento, invece, il processo mantiene una dispersione compatibile con quella del limite continuo.

L’idea centrale del teorema è che, aumentando il numero di suddivisioni dell’intervallo temporale, la Random Walk scalata perde progressivamente il suo carattere puramente discreto e assume un andamento globale sempre più simile a quello di una traiettoria browniana.

Il risultato è importante perché mostra una proprietà di invarianza: il limite non dipende dalla distribuzione specifica dei passi, purché siano soddisfatte tre condizioni fondamentali: indipendenza, media zero e varianza finita.

In modo intuitivo, il principio di Donsker estende l’idea del teorema centrale del limite dal caso delle singole somme al caso dell’intero processo stocastico. Per questo motivo, la Random Walk opportunamente riscalata costituisce un modello discreto naturale del Moto Browniano Aritmetico.
